# Public Frontend for Jackie’s Supper Club Menu Form

This folder contains only the static frontend files needed for the menu selection form. All backend and sensitive configuration files are kept private and are NOT published here.

## Files
- form.html
- form.js
- form.css
- formConfig.js
- test.html (optional, for local testing)

## Do NOT include
- Any .gs (Apps Script) files
- Config.gs (contains secrets and sheet IDs)
- Any backend or deployment files
- Any private customer data

## Hosting
This folder is safe to use as the root for GitHub Pages or any static host.
